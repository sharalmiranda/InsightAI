# 🧠 InsightCRM AI
**Predictive Customer Intelligence & Retention Engine**

InsightCRM AI is a modern, single-page dashboard designed to help businesses turn raw customer data into actionable retention strategies. It analyzes purchase history and engagement signals to predict churn risk, identify high-value customers, and instantly generate personalized outreach campaigns.

## ✨ Key Features
* **Predictive Value Scoring:** AI-computed health scores (0-100) based on purchase frequency, recency, and spend patterns.
* **Smart Customer Segmentation:** Automatically classifies customers into dynamic tiers (e.g., VIP Champion, Regular, At-Risk).
* **Behavioral Intelligence Signals:** At-a-glance metrics for Purchase Cadence, Brand Loyalty, and Churn Probability.
* **AI-Recommended Engagement:** Suggests the highest-converting action for each specific user (e.g., Win-back campaigns, VIP upgrades).
* **AI Outreach Composer:** Instantly generates hyper-personalized, ready-to-send messages calibrated to the customer's exact profile and predicted intent.
* **Interactive UI:** Smooth, state-driven dashboard that updates instantly when selecting different customer profiles.

## 🛠️ Tech Stack
This project was built entirely without heavy frameworks to ensure lightning-fast performance and zero-dependency setup.
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, Flexbox/Grid layouts, and smooth transitions (No external UI libraries used).
* **Vanilla JavaScript (ES6):** State management, DOM manipulation, and dynamic data rendering.

## 🚀 How to Run Locally
Because this project uses native web technologies, there are no build steps, node modules, or local servers required.

1. **Download or Clone** this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to open it in any modern web browser (Chrome, Safari, Firefox, Edge).
